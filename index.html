Ogre

Clojure library for querying Blueprint graphs (Gremlin dialect)

Get Started! » Read doc guides » Join the Mailing List » Contribute »

Powerful

Ogre provides a very expressive and efficient way of querying Blueprint graphs

, for example, Titan or Neo4J Blueprints implementation.

Efficient

Very low overhead over the Gremlin Java implementation

Fun

With Ogre, querying graphs is fun. No more hair pulling!
